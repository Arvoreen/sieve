Any of the following header ...
To From Cc Bcc Reply-To
... any of the keyword(s):

Compares headers against E-Mail addresses.

You typically use test with headers like "to", "from", "cc" etc.

As this test is aware of e-mail addresses containing display names. A header containing '"roadrunner" <roadrunner@acme.example.com>' is considered to be equivalent to "'roadrunner@acme.example.com"

If the header should be matched against a string use the header test.
