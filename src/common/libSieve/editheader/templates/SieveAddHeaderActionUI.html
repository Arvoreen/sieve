Adds a new header to the meassage. The existing message header are untouched. By default, the header field is inserted at the beginning of the existing message header.

Note: the header name is by definition a case insensitive 7bit US-ASCII string. Leading and tailing whitespaces are ignored. Headers with invalid names are silently ignored.
